Restgo

Booking a pool villa

Conducted user research and designed a pool villa booking system, including search, booking flow, and responsive cross-platform components.
Overview
Restgo is an online platform that allows users to search and book private villas and vacation homes in Thailand. The main objective of this project is to improve the user experience by simplifying the accommodation search process and making the booking journey more intuitive.
Problem
Users who are planning group trips or vacations often struggle to quickly find suitable villas that match their needs, such as location, capacity, and amenities. If the browsing and booking process is not clear, users may abandon the website before completing a reservation.
Pain Points
- Difficult to quickly compare villas
- Users need to open many pages to see details
- Booking process may require multiple steps
Goal
The goal of this design is to help users find and book villas faster by simplifying the search experience and highlighting key information.
UX Goals
- Improve villa discovery
- Simplify the booking process
- Help users compare accommodation options quickly
Design Solution
To improve usability, the interface highlights search functionality and presents villa listings in a clear card layout. Key information such as price, location, and capacity is displayed prominently so users can quickly compare accommodation options.
UX Improvements
- Improved search visibility
- Clear villa card design
- Simplified booking flow
Result / Impact
The improved interface helps users find suitable villas more efficiently and complete bookings with fewer steps, resulting in a smoother and more intuitive user experience.
